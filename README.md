Libraries are designed to work on KiCad V10

How to add this library to your KiCad V10:

Clone this github repository to your machine

In the schematic editor, go to Preferences > manage symbol libraries 

Add a custom Global Library with the nickname 1_Custom_Library and set the library path to be the folder you cloned the github repo to 

Now switch to the PCB editor, go to preferences > manage footprint libraries 

Add a custom Global Library with the nickname 1_Custom_Library and set the library path to be the .pretty file located in the github repo
